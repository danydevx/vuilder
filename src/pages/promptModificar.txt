- Uso Vue 3 `<script setup>` y `<style lang="less" scoped>`.
- Props estándar:  
  `id`, `scheme`, `title`, `subtitle`, `backgroundImg`, `enableOverlay`,

- Si hay dos columnas, `order="inverted"` invierte la posición.  
- Uso `schemeClass = computed(() => 'component-scheme-' + props.scheme)` para aplicar color tokens.  
- Todas las secciones deben respetar mis **design tokens** (variables CSS como `--surface`, `--text`, `--accent`, `--text-muted`, `--accent-contrast`, etc.).  
- Efectos con **AOS** se controlan con la prop `animate` (si es `false`, no agrega `data-aos`).  
- Siempre incluir `<style lang="less" scoped>` con colores, sombras y espaciados basados en mis tokens.
- Fondo y overlay con `pointer-events: none` y `z-index` bien definidos.  
- `fluid` controla si el contenedor es `.container` o `.container-fluid px-3 px-md-5`.
- Respetar mis componentes previos (`SectionSocial1–7`, `SectionFaqChat`, `SectionFaqs5`, `SectionCalendar2`, etc.) como guía visual.

🎨 **Estilo visual:**
- Moderno, limpio, elegante, móvil primero.  
- Usa `rounded-4`, `shadow-sm`, `shadow-lg` donde aplique.  
- Animaciones sutiles con `AOS` y `color-mix` para hovers.
- Evita colores adicionales; usa los tokens del esquema activo.

⚙️ **Librerías disponibles:**
- `Swiper`, `GLightbox`, `AOS`, `Bootstrap Icons`.

✨ **Tu tarea:**
Cuando te pida una nueva sección o modificar una existente, genera el **código completo del componente Vue** (template + script + style) siguiendo estas convenciones.
